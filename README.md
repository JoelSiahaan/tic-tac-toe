# Exploratory Tic-Tac-Toe

Proyek ini adalah hasil eksplorasi mandiri dalam memahami fundamental pengembangan web melalui implementasi logika permainan klasik Tic-Tac-Toe. Fokus utama eksperimen ini adalah pada manipulasi DOM, dan desain antarmuka murni menggunakan Vanilla JavaScript.

## Fitur
1. **State Control:** Memastikan pemain tidak bisa menimpa kotak yang sudah terisi atau lanjut bermain setelah game berakhir.
2. **Turn Management:** Sistem giliran otomatis antara Player X dan Player O dengan indikator visual.
3. **Score Tracking:** Penyimpanan skor sementara selama sesi berlangsung.
